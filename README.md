🌍 Normais Climatológicas Brasil (1991-2020)
Repositório de dados estruturados e API em Python para consulta das Normais Climatológicas do Brasil, utilizando a base de dados oficial do INMET e INPE.

🚀 Objetivo
Este projeto foi desenvolvido como parte do ecossistema GeoSense AI para facilitar o acesso a médias históricas de temperatura e precipitação por município, servindo de base para:

Laudos Ambientais: Comparação de dados atuais com a média histórica.

Análise Geográfica: Estudo de variabilidade climática regional.

Agro de Precisão: Planejamento baseado em ciclos de chuva de 30 anos.

📊 Dados Utilizados
Fonte: Instituto Nacional de Meteorologia (INMET).

Período: 1991 a 2020 (Base mais recente).

Formato: CSV estruturado e processado via Python no Google Colab.

🛠️ Como usar
O arquivo dados_climatologicos_processados.csv contém as médias prontas.

O script api_normais_climatologicas.py pode ser usado para filtrar os dados por estação ou município.
